### Windows Runtime projections (implicit)

`PodBridge.App` and `PodBridge.Windows` target `net10.0-windows10.0.22621.0`,
which implicitly references the Windows SDK .NET projections generated by C#/WinRT
(CsWinRT) — `Microsoft.Windows.SDK.NET.Ref` — used to reach the WinRT BLE and
audio APIs. These projections are published by Microsoft under the **MIT** license
(https://github.com/microsoft/CsWinRT). They are part of the Windows targeting
pack rather than an explicit `PackageReference`.
